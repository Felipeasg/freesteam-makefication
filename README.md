Dependencies
============  

GSL
---  

```
$ wget ftp://ftp.gnu.org/gnu/gsl/gsl-2.5.tar.gz
$ tar -xvf gsl-2.5.tar.gz
$ cd gsl-2.5
$ ./configure --prefix=/opt/apps/gsl/2.5-gcc-5.4.0 CFLAGS=-fPIC
$ sudo make install
```
